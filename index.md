# Política de Privacidade — MegaMoment Lotto

**Última atualização:** 23 de abril de 2026

Este documento descreve como o aplicativo MegaMoment Lotto ("o app") trata as informações dos usuários.

---

## 1. Coleta de Dados

O MegaMoment Lotto **não coleta, armazena nem transmite dados pessoais** para servidores externos. Todas as informações geradas (números, grupos, datas) ficam armazenadas exclusivamente no dispositivo do usuário via armazenamento local.

---

## 2. Dados de Uso e Publicidade

O app utiliza o **Google AdMob** para exibição de anúncios. O AdMob pode coletar dados de uso e identificadores de dispositivo para personalização de anúncios, conforme a [Política de Privacidade do Google](https://policies.google.com/privacy).

O usuário pode optar por não receber anúncios personalizados nas configurações do dispositivo Android em: **Configurações > Google > Anúncios**.

---

## 3. Permissões

O app solicita apenas a permissão de **INTERNET**, necessária para exibição de anúncios via AdMob. Nenhuma outra permissão é utilizada.

---

## 4. Dados de Crianças

O MegaMoment Lotto não é direcionado a crianças menores de 13 anos e não coleta conscientemente dados de menores.

---

## 5. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Alterações serão refletidas na data de "última atualização" acima.
